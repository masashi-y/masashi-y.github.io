# Lesson 9

## 9.1. Irregularities in conjugation

### 1st conj. verbs

- irregular（語根に母音を含まないもの）の数はかなり限られるが、そのうちのいくつかはよく使われる
- irregular動詞のうち、語尾が-ამ, -ავ, -ი（-იに関しては一部）のものは以下の*strong aorist series*語尾をとる
  - aorist: -ი, -ა, -ეს
  - optative: -ა, -ას, -ან

#### -ამ 動詞

- およそ９つこのような語根が存在する
- これらはaorist/optativeにstrong語尾をつける

| Future                   | Aorist                        | Optative                       |
| ------------------------ | ----------------------------- | ------------------------------ |
| მიაბამს (tie)            | მი(ვ)აბი(თ), მიაბა, მიაბეს    | მი(ვ)აბა(თ), მიაბას, მიაბან    |
| დადგამს (put)            | და(ვ)დგი(თ), დადგა, დადგეს    | და(ვ)დგა(თ), დადგას, დადგან    |
| დაასხამს (pour)          | და(ვ)ასხი(თ), დაასხა, დაასხეს | და(ვ)ასხა(თ), დაასხას, დაასხან |
| ჩაიცვამს (dress oneself) | ჩა(ვ)იცვი(თ), ჩაიცვა, ჩაიცვეს | ჩა(ვ)იცვა(თ), ჩაიჩვას ჩაიცვან  |

#### -ავ 動詞

- およそ１２個
- aoristはstrong語尾をつけ、１，２人称の場合語根に-ა-を挿入する
- optativeはstrong語尾をつけ、母音を含まない語根から作る

| Future         | Aorist                      | Optative                   |
| -------------- | --------------------------- | -------------------------- |
| მოკლავს (kill) | მო(ვ)კალი(თ), მოკლა, მოკლეს | მო(ვ)კლა(თ), მოკლას მოკლან |
| ჩაფლავს (bury) | ჩა(ვ)ფალი(თ), ჩაფლა, ჩაფლეს | ჩა(ვ)ფლა(თ), ჩაფლას ჩაფლან |

#### -ი 動詞 (strong語尾)

- およそ１５個
- そのうち３つの語根は-ნで終わり、その場合strong語尾をとる
- aoristはstrong語尾をつけ、１，２人称の場合語根に-ე-を挿入する
  - 位置は語根最後の子音の前
- optativeはstrong語尾をつけ、母音を含まない語根から作る
  - მოჭრისはნ語尾ではないが例外的にこれに従う

| Future           | Aorist                        | Optative                       |
| ---------------- | ----------------------------- | ------------------------------ |
| შექმნის (create) | შე(ვ)ქენი(თ), შექმნა, შექმნეს | შე(ვ)ვმნა(თ), შექმნას, შექმნან |
| მოჭრის (cut)     | მო(ვ)ჭერი(თ), მოჭრა, მოჭრეს   | მო(ვ)ჭრა(თ), მოჭრას, მოჭრან    |

#### -ი 動詞 (regular語尾)

- およそ２０個
- 普通のaorist/optative語尾をとるが、すべての形で-ა-が語根に挿入されるのが例外的
  - 位置は最初の子音、harmonic cluster, 子音 + vのうしろ

| Future                   | Aorist                           | Optative                          |
| ------------------------ | -------------------------------- | --------------------------------- |
| ჩათვლის (consider)       | ჩა(ვ)თვალე(თ), ჩათვალა, ჩათვალეს | ჩა(ვ)თვალო(თ), ჩათვალოს, ჩათვალონ |
| აღზრდის (educate, raise) | აღ(ვ)ზარდე(თ), აღზარდა, აღზარდეს | აღ(ვ)ზარდო(თ), აღზარდოს, აღზაღდონ |

- 他：დასჯის (punish), დაშლის (unwrap, spread out), გამოცდის (test), გამოცვლის (change), გადაიხდის (pay)

### 2nd conj. verbs

- irregularな1st conj.動詞から派生した2nd conj.動詞について
- aorist/optativeは1st conj.動詞と以下を除き全く同じ形をとる
  - ი-マーカーをつけ、aorist３人称複数は-ეს語尾の代わりに-ნენをつける

| 2nd conj.                                | Aorist                               | Optative                             |
| ---------------------------------------- | ------------------------------------ | ------------------------------------ |
| მიაბამს -> aor. და(ვ)აბი(თ), და(ვ)აბა(თ) | და(ვ)იბი(თ), დაიბა, დაიბნენ          | და(ვ)იბა(თ), დაიბას, დაიბან          |
| ჩათვლის -> aor. ჩა(ვ)თვალე(თ)            | ჩა(ვ)ითვალე(თ), ჩაითვალა, ჩაითვალნენ | ჩა(ვ)ითვალო(თ), ჩაითვალოს, ჩაითვალონ |

#### Root 2nd conj. verbs

- "root": aorist/optativeにおいて、2nd conj.をマークするი-, -დのどちらも存在しないもの
- ほとんど（２０ちょっと）のroot 2nd conj.動詞は対応する1st conj.がない
- 辞書項目は３人称単数未来形で、現在/未来のscreevesでは普通の2nd conj.として語形変化する
- aorist/optativeでは、-დ語尾の2nd conj.動詞と同じ語尾をつける
  - aorist: -ი, -ა, -ნენ
  - optative: -ე, -ეს, -ნენ
- 例：მო-კვდ-ებ-ა (die), შე-ცდ-ებ-ა (err), შე-ხვდ-ებ-ა (encounter, meet), გა-ხდ-ებ-ა (become), და-რჩ-ებ-ა (remain)

| Future   | Aorist                         | Optative                        |
| -------- | ------------------------------ | ------------------------------- |
| მოკვდება | მო(ვ)კვდი(თ), მოკვდა, მოკვდნენ | მო(ვ)კვდე(თ), მოკვდეს, მოკვდნენ |

- 名詞は-ომაをつけて作ることが多い
  - მოკვდომა, შეცდომა, მოხდომა
  - そうでないもの：დარჩენა, შეხვედრა
- root 2nd conj.動詞には、ა- ... -ობの1st conj.から派生した2nd conj.を含む。

| 1st conj.         | 2nd future | 2nd aorist |
| ----------------- | ---------- | ---------- |
| გაათბობს (heat)   | გათბება    | გათბა      |
| გააშრობს (dry)    | გაშრება    | გაშრა      |
| გამოაცხობს (bake) | გამოცხვება | გამოცხვა   |

- root 2nd conj.動詞のrelativeな用法は少ないが、そのような場合はu-seriesをつける
- 例：
  - გასულ წელს ბებია მოკვდა.
    - Last year grand mother died.
  - გასულ წელს ბებია მომიკვდა.
    - Last year grandmother died on me (my grandmother died)
- **位置の変化を記述する動詞**
  - დადგება (stand up), დაწვება (lie down), დაჯდება (sit down sg.), დასხდებიან (sit down pl.)
    - 名詞：ადგომა, დაჯდომა, დასხდომა, დაწოლა
  - これらは、１，２人称aorist以外はroot 2nd conj.と同じ変化をする

| 1, 2 aorist  | aorist 3sg.    | optative 3sg.  |
| ------------ | -------------- | -------------- |
| და(ვ)დექი(თ) | დადგა          | დადგეს         |
| და(ვ)წექი(თ) | დაწვა          | დაწვეს         |
| და(ვ)ჯექი    | დაჯდა          | დაჯდეს         |
| და(ვ)სხედით  | დასხდნენ (pl.) | დასხდნენ (pl.) |

- 歴史的にaoristはდადეგ -> დადექ -> დადექიという変化をたどったらしい
- დაჯდებაとდასხდებიანの使い分けは必須
  - 文法的に単数だが複数の人物を指す場合は特別にდასხდებაをつかう
- 例：
  - ივანე შემოვიდა და დაჯდა.
  - ივანე და მარიამი შემოვიდნენ და დასხდნენ.
  - ჩემი მშობლები შემოვიდნენ და დასხდნენ.
  - ჩემი სამი მეგობარი შემოვიდა და დასხდა.
  - ახალგაზრდა წყვილი შემოვიდა და დასხდა. 
- **位置を記述する動詞**
  - 上の４つに対応して位置を記述する動詞があるが例外的な挙動をする
  - 語末にყოფნაの変化系をつける
  - imperfectとpresent stem conjunctiveはない

|      | be standing  | be lying down | be sitting sg. | be sitting pl. |
| ---- | ------------ | ------------- | -------------- | -------------- |
| 1st  | ვდგავარ(თ)   | ვწევარ(თ)     | ვზივარ         | ვსხედვართ      |
| 2nd  | დგახაო(თ)    | წევხაო(თ)     | ზიხარ          | სხედხართ       |
| 3rd  | დგას, დგანან | წევს, წვანენ  | ზის (სხედს)    | სხედან         |

- 未来形はroot 2nd conj.動詞と同じ変化をする (ი-をつける)
- aoristでは、位置変化の動詞同様の子音変化をたどる

| Future    | Aorist                    |
| --------- | ------------------------- |
| იდგება    | (ვ)იდექი(თ), იდგა, იდგნენ |
| იწვება    | (ვ)იწექი(თ), იწვა, იწვნენ |
| იჯდება    | (ვ)იჯექი, იჯდა, (ისხდა)   |
| ისხდებიან | (ვ)ისხედით, ისხდნენ       |

- 主語はnominativeであり、verbal nounは位置変化動詞と同じものをつかう

### 例外的動詞 იცის

- 名詞：ცოდნა
- present seriesではტირისと同じ変化 (8.1)
  - 例外的なのは、present seriesでも主語が能格、目的語が主格であることである
  - このような動詞はこれだけ
- 未来形は12.1
- aoristはなしで、代わりにimperfect იცოდა を使う
- optativeもなく、代わりにconjunctive present იცოდეს を使う

## 9.2. Personal pronouns

- これらは基本的に強調の目的に使われる

|      | sg.           | pl.   |
| ---- | ------------- | ----- |
| 1st  | მე (gen. ჩემ) | ჩვენ  |
| 2nd  | შენ           | თქვენ |

| 3rd    | sg.     | pl.          |
| ------ | ------- | ------------ |
| nom.   | ის, იგი | ისინი, იგინი |
| erg.   | მან     | მათ          |
| dat.   | მას     | მათ          |
| gen.   | მის     | მათ          |
| instr. | მით     | -            |

- 注意
  - 例外的にმასに-ში, -ზეをつけるときსを落とさない
    - მასში, მასზე
    - cf. demonstrative pronouns: იმაში, იმაზე
- 他の後置詞と代名詞の組み合わせ
  - -თვის: ჩემთვის, შენთვის, მისთვის, ჩვენთვის, თქვენთვის, მათთვის
  - -გან: ჩემგან, შენგან, მისგან, ჩვენგან, თქვენგან, მათგან
  - -თან: ჩემთან, შენთან, მასთან, ჩვენთან, თქვენთან, მათთან
  - などなど
- შენ, თქვენはvocativeとしても使える
  - これ用の特別な形შე, თქვეも存在する
    - შე გიჟო! (you lunatic you!)
    - თქვე სულელებო! (you fools you!)

## 9.3 Comparison of adjectives and adverbs

比較は、(1) უფრო (more) + ვიდრე (than)か、(2) -ზეのどちらかで表現する。

1. ქართული ანბანი უფრო ძველია, ვიდრე სლავური.
2. ქართული ანბანი სლავურზე ძველია.

最上級はყველაზე (than all)と形容詞（副詞）をつなげるだけ

- ყველაზე დიდი მთა ევროპაში იალბუზია.
  - The highest mountain in Europe is Elbrus.

いくつかの形容詞には比較級、最上級に対応する形が存在する

|       | 比較級  | 最上級                 |
| ----- | ------- | :--------------------- |
| კარგი | უკეთესი | საუკეთესო              |
| ცუდი  | უარესი  | უველაზე ცუდი (უარესი)  |
| ცოტა  | ნაკლები | ყველაზე ცოტა (ნაკლები) |
| ბევრი | მეტი    | ყველაზე ბოვრი (მეტი)   |
